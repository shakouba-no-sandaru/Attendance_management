# 出退くん（Time Attendance Management System）

## ■Overview
従業員の出退勤を管理するために開発したWebアプリケーション 
フロントエンドからバックエンド、データベースまで一貫して設計・実装

---

## ■Features
- 認証機能（ログイン）
- 勤怠データの登録 / 編集 / 削除
- 月別の勤怠一覧表示
- MySQLによるデータ管理

---

## ■Tech Stack
- Frontend：Vue3 / TypeScript  
- Backend：Node.js / Express  
- Database：MySQL  

---

## ■Architecture
フロントエンド・バックエンド・データベースを分離した構成で設計  
APIを介したデータ連携を実装

---

## ■Highlights
- フルスタックでのアプリケーション開発経験
- 実務を意識したデータ設計・API設計
- 保守性を考慮したシンプルな構成
